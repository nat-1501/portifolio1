    HTML5

    HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

    CSS

    CSS3 é a terceira mais nova versão das famosas CSS, pela qual se define estilos para um projeto web. Com efeitos de transição, imagem, imagem de fundo/background e outros.

    PHP

    PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.

    Bootstrap

    O Bootstrap nada mais é que um framework Front End utilizados por milhares de desenvolvedores web pelo mundo. A partir dele, muitas etapas do desenvolvimentos web se tornam mais rápidas e dinâmicas, pois já trazem consigo diversos elementos prontos e estilizados.

    Git e Git-Hub

    GIT é um sistema de controle de versão de código distribuído, open source e gratuito, enquanto que o GITHUB é uma plataforma de desenvolvimento em que você pode armazenar códigos, gerenciar seus projetos. Ferramentas que são usadas para facilitar a vida de um programador.

    Java Script

    JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível. untamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.
